Run Hydris as a Home Assistant app/add-on.

Hydris is an open-source coordination engine for sensors, assets, mission systems,
and large-area situational awareness networks.

[:books: Read the full app/add-on documentation][docs]

## Installation

Add this repository to Home Assistant as a custom app/add-on repository, then
install the Hydris app/add-on.

## Support

Open an issue in this repository for packaging issues. For Hydris itself, see
the upstream project.
